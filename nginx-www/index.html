In-browser examples

These are C++ examples compiled to Emscripten. All the packing happens automatically in CMake scripts, no extra work, same process as creating iOS applications, Snap packages, AppImages and Universal Windows Platform containers.

These are simple code tests, and don't have much in the way of interactivity.

  • Environment information: Prints storage paths in the virtual filesystem provided by Emscripten
  • Basic OpenGL: A basic OpenGL scene with a faulty emulation of 2D texture arrays

Statistics

These numbers are collected from devices sending runtime reports to `coffee.birchtrees.me`. This is made possible by an embedded profiler in the application, activated by specifying the following environment variable:

COFFEE_REPORT_URL=https://<server>/<endpoint> <program>

or on Android builds, this is done by specifying an activity intent:

adb shell am start -n <com.package/.Activity> --es COFFEE_REPORT_URL <report-server>
